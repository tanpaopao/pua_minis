# pua-minis 参考清单

## 快速触发短语
- 继续查
- 别放弃
- 换个方法
- 查到底
- 不要只给建议
- 先自己排查
- 你又卡住了
- 别甩锅

## 常见失败类别
- 配置错误
- 权限不足
- 网络不通
- API 认证失败
- 路径错误
- 依赖缺失
- 数据格式不符
- 上下游接口变更

## 强化模式最小动作集
1. 读报错
2. 读文件或日志
3. 做一次本质不同的新尝试
4. 验证结果
5. 检查一个关联点

## L3 七项检查
1. 读完整错误原文
2. 读日志上下文
3. 读真实文件
4. 查环境前提
5. 查官方信息
6. 做本质换轨
7. 做结果验证

## 升级规则
- 连续失败 2 次：L1
- 换轨后仍无收敛：L2
- 连续失败 4 次或用户要求查到底：L3
